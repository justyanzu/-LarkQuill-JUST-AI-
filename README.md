  技术栈：Python · MCP 协议 · GPT-Image-2 · lark-cli  · OpenClaw AI Agent                                                                                
                                                                                                                                                                     
  功能：                                                                                                                                                             
  • 自动读取飞书多维表格中的商品信息（名称/价格）

    生成电商618风格大促海报                                                                                                                                          
  • 将生成的促销图自动上传回多维表格对应记录
