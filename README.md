项目：雀翎 (LarkQuill) — 飞书多维表格 AI 电商自动化助手                                                                                                            
                                                                                                                                                                     
  技术栈：Python · MCP 协议 · GPT-Image-2 · lark-cli · Grsai 中转 · OpenClaw AI Agent                                                                                
                                                                                                                                                                     
  功能：                                                                                                                                                             
  • 自动读取飞书多维表格中的商品信息（名称/价格）                                                                                                                    
  • 通过 MCP JSON-RPC 协议调用 AI 图片生成服务，为每个商品                                                                                                           
    生成电商618风格大促海报                                                                                                                                          
  • 将生成的促销图自动上传回多维表格对应记录                                                                                                                         
  • 全流程一键触发，无需人工介入
